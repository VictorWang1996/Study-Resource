# 题目描述
写一个函数，求两个整数之和，要求在函数体内不得使用+、-、*、/四则运算符号。

# 解法
不能用四则运算——》位运算
异或：计算加法
(num1&num2)<<1 ——》用于计算进位
temp + num2 循环计算
直到进位为0为止

```java
public class Solution {
    public int Add(int num1,int num2) {
        while (num2!=0) {
            int temp = num1^num2;//加法
            num2 = (num1&num2)<<1;//计算进位
            num1 = temp;
        }
        return num1;
    }
}
```
